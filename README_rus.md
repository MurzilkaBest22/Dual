ОПИСАНИЕ
Dual – полная библиотека автоматического дифференцирования на основе дуальных и гипердуальных чисел. Она включает в себя:
Вычисление полных, частных и смешанных производных любых натуральных порядков.
Вычисление градиентов высших порядков (включая гессианы).
Поддержку дробных производных Римана‑Лиувилля и Капуто.
Работу с рациональными числами (при использовании библиотеки Rational).
Специальные функции (гамма‑, бета‑, дзета‑функции Римана).
Классы для дуальных комбинаций и гипердуальных комбинаций.

ВОЗМОЖНОСТИ
Классы: Dual, DualCombination, HyperDualCombination.
Операторы дифференцирования для различных типов аргументов.
Гибридный подход для дробных производных (автоматическое дифференцирование + численное интегрирование).
Полная поддержка ОДЗ для всех функций.

УСТАНОВКА И СБОРКА
Библиотека состоит из нескольких файлов:
dual.h, dual.cpp - дуальные числа
multidual.h, multidual.cpp - дуальные комбинации
hyperdual.h, hyperdual.cpp - гипердуальные комбинации
special_functions.h, special_functions.cpp - специальные функции
fractional.h, fractional.cpp - дробные производные
alldiff.h, alldiff.cpp - подключаемый файл

ДЛЯ ИСПОЛЬЗОВАНИЯ
Скопируйте папку Dual в ваш проект.
Добавьте все .cpp файлы (dual.cpp, special_functions.cpp, multidual.cpp, hyperdual.cpp, fractional.cpp, alldiff.cpp) к компиляции.  
Включите основной заголовок:

#include "Dual/alldiff.h"

(этот файл подключает все остальные).

Используйте пространство имён dual.

Если вы не хотите использовать все функции, можно подключать только нужные заголовки.

ЗАВИСИМОСТИ
RealUtils (обязательно)
Rational (для работы с рациональными числами и дробными порядками)
Библиотека Boost.Math (для специальных функций и численного интегрирования). Убедитесь, что Boost установлен и доступен.

ПРИМЕР

#include <iostream>
#include "Dual/alldiff.h"

auto f = [](dual::Dual x) {
    return dual::sin(x) * dual::exp(x);
};

int main() {
    double x = 1.0;
    double df = dual::D(f, x);
    std::cout << "f'(1) = " << df << std::endl;
    return 0;
}
